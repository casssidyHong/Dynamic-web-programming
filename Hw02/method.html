Explanation of Calculations

Concept Definition Calculation Method Example
Maximum (Max) The largest number in a given set. Compare each number in the set and find the largest one. For [1, 3, 7, 5], the max is 7.
Minimum (Min) The smallest number in a given set. Compare each number in the set and find the smallest one. For [1, 3, 7, 5], the min is 1.
Median The middle number when the numbers are sorted in order. Traverse through the numbers as a string and use a custom function to insert each number into a sorted string.
After inserting all numbers into the sorted string, find the middle two numbers and calculate their average.
If the number of elements is odd, take the middle number directly.
If the number of elements is even, average the two middle numbers. 		For input "3,1,4,2,8,6,7,5",
- Insert each number to get sorted string: "1,2,3,4,5,6,7,8".
- For an even number of elements, the median is calculated as (4+5)/2 = 4.5.
- So, the median is "4.5" (as a string).
Prime Number A number that has only two divisors: 1 and itself. Check if the number is divisible only by 1 and itself without remainder. 2, 3, 5, 7 are prime numbers.
Armstrong Number A number that is equal to the sum of its own digits each raised to the power of the number of digits. Separate each digit, raise it to the power of the number of digits, and sum them up to see if it equals the original number. 153 = 1^3 + 5^3 + 3^3, so 153 is an Armstrong number.
Fibonacci Number A number that is part of the sequence where each number is the sum of the two preceding ones. Start with 1 and 1, then sum the last two numbers to get the next number in the sequence. 1, 1, 2, 3, 5, 8, 13, etc.
